# 什么是LLM

LLM（Large Language Model）基于自然语言处理（NLP），使用大量文本数据作为训练样本，能够生成和理解人类语言。

# LLM能做什么

## 文本生成

文本创作、摘要与改写、结构化生成

## 自然语言理解与分析

语义分析

- 情感分析（评论、反馈）
- 意图识别（客服对话、用户查询）
- 实体提取（人民、地点、事件）

知识推理

- 逻辑问题解答（数学、常识推理）
- 因果分析（事件关联性）

## 代码相关

代码生成、debug、技术文档处理

# 推理模型与通用模型

| 维度       | 通用模型                             | 推理模型                                             |
| ---------- | ------------------------------------ | ---------------------------------------------------- |
| 擅长领域   | 规范性任务（文本生成、自然语言处理） | 逻辑推理类（数学推到、代码生成、逻辑分析、复杂问题） |
| 处理速度   | 快速响应                             | 稍慢                                                 |
| 规范性     | 强规范约束                           | 若规范约束                                           |
| 结果导向   | 目标确定性高（结果可预期）           | 目标开放性高（结果多样性）                           |
| 风险特征   | 稳定可控                             | 不确定性高                                           |
| 路径灵活性 | 线性路径（流程标准化）               | 网状路径（多路径探索）                               |

两种模型的提示语差异

通用模型：过程-结果清晰（指令），需要清晰定义你是谁、你的目标是什么，并可能需要给出具体的执行步骤；

推理模型：目标清晰，结果可以模糊（推理），强调准确表达目标而非具体步骤，让AI自行决定最佳行动路径

## 通用模型-RTGO提示语结果

Role（角色）；定义AI的角色；

Task（任务）：具体任务描述；

Goal（目标）：期望达成什么目标效果；

Objective（操作要求）：字数要求、段落结构、用词风格、内容要点、输出格式；

```
Role：你是一位经验丰富的营销专家，擅长撰写吸引人的产品推广方案
Task：请为即将发布的新产品“智能健身手环”撰写一份推广文案。该产品具有心率监测、部署统计等功能。
Goal：通过文章吸引潜在客户，提高产品的知名度和销售量
Objective：
- 文案长度：不超过500字
- 内容要点：介绍产品的核心功能、优势
- 输出格式：以Markdown格式输出
- 用词风格：简洁明了，富有吸引力
```

## 推理模型-提示语技巧

不要过多干预，不用手把手教学。

```
[角色设定] （可选） 例如：“你是一个严谨的数学老师”或“你是一个逻辑清晰的侦探”。

[背景] 清晰、无歧义地陈述你需要解决的问题和所有已知信息。

[核心指令] 例如：“请针对以下问题，进行一步步的推理。”

[推理步骤框架] （可选，用于极复杂问题） 例如：
1. 首先，提炼问题的核心。
 2. 其次，分解已知条件。
 3. 然后，寻找条件和目标之间的逻辑关系。
 4. 最后，得出结论。

[输出格式要求] （可选） 例如：“请将你的最终答案用【答案】：...的格式框起来。”
```

# AI幻觉

简单来说就是一本正经的胡说八道。

形成原因：由于模型在缺乏相关信息的情况下，通过概率性选择生成内容，而非基于真实世界的知识库或逻辑推理，使得其输出不仅难以信赖，且可能误导用户。

## 降低幻觉的方式

1、联网搜索

2、双AI验证/大模型协作：利用deepseek生成答案后，再用其他大模型进行审查，相互监督，交叉验证

3、提示词工程

1）知识边界限定：通过时空维度约束降低虚构可能性（**本质：约束大模型**）

- 时间锚定法：“基于2023年之前公开的学术文献，分步骤解释量子纠缠现象”（规避未来时态虚构）
- 知识锚定法：”基于《中国药典》回答，xxxx“（限定权威来源）
- 领域限定法：“作为风控运营专家，xxx”（添加专业身份）
- 上下文提示：“根据《2024全球能源转型报告》显示：2030年光伏发电成本降低至0.2美元/千瓦时，xxx，请基于此数据分析中国光伏基地发展额三个挑战”（嵌入权威数据片段）

2）对抗性提示：大模型自我审查

- 链式验证：先让模型给出观点，然后列出支撑观点的权威数据源，检查是否有矛盾信息
- 预设验证条件，迫使模型交叉检查信息：先回答问题答案，然后让大模型从以下角度验证答案可靠性：物理学界主流观点、近五年相关论文数量。

# 大模型常见参数

**Temperature：**调整生成的随机性。温度越低，生成的内容越相同；温度越高，生成内容越随机。

**Top-K：**限制候选词数量，使生成内容集中。

**Top-P：**过滤掉累积概率低于指定阈值（p）的词，从而增加输出的多样性。

**Frequency Penalty：**值越大，词重复越少。

对于创意写作，可以使用较高的 temperature与较低的 Top-K/Top-P；

对于要精确的分析人物，可以选择较低的 temperature 与较高的 Top-K/Top-P；
